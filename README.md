# erp-sales-data# ERP Sales Analytics Project

## 📌 Project Overview
This project is an end-to-end Data Analytics pipeline. It transforms raw ERP sales data into a structured relational database using **SQL** and visualizes key business performance metrics through an interactive **Power BI Dashboard**.

## 🛠 Skills Showcased: SQL Database Engineering
*   ⚙️ **Database Schema Design:** Architected a relational database model by defining `items`, `customers`, and `sales` tables, ensuring data integrity through the implementation of **Primary Keys** and **Foreign Key constraints**.
*   🧹 **Data Cleaning & Normalization:** Performed robust data cleaning to handle anomalies, including standardizing date formats and managing missing values to prepare the dataset for complex analytical queries.
*   📊 **Complex Query Formulation:** Developed advanced SQL queries, including multi-table **JOINs**, **Aggregations**, and **Window Functions**, to derive actionable business insights such as revenue trends, customer segmentation, and product performance.
*   📈 **Business Intelligence Readiness:** Optimized data extraction processes, ensuring the dataset was structured effectively for seamless integration into Power BI for automated dashboarding and performance tracking.
